# Layer 0 lesson roadmap

This roadmap defines the planned lesson sequence for Chapter 00: Layer 0 — Mental Models & Computing Foundations. It is intentionally limited to lesson titles and module structure; individual lesson content will be added in future changes.

Layer 0 aligns with [Blueprint v0.1](../BLUEPRINT-v0.1.md#00-layer-0--mental-models--computing-foundations) and prepares learners to follow the idea-to-production flow described in the [System Map](../SYSTEM-MAP.md).
