# Layer 0 resources

This page defines the resource categories for Chapter 00: Layer 0 — Mental Models & Computing Foundations. Resources should support the chapter scope in [Blueprint v0.1](../BLUEPRINT-v0.1.md#00-layer-0--mental-models--computing-foundations) and reinforce the end-to-end view in the [System Map](../SYSTEM-MAP.md).

Future additions should prioritize durable, beginner-friendly material that strengthens mental models without turning this chapter into a vendor-specific tool guide.

## Books

Books in this section should explain computing foundations, operating systems, networking, software design, or production-oriented engineering concepts at an introductory or intermediate level.

## Official Documentation

Official documentation in this section should come from standards bodies, language projects, operating system projects, cloud-neutral foundations, or maintainers of widely used open source systems.

## Interactive Labs

Interactive labs in this section should help learners observe foundational behavior directly, such as process execution, networking, HTTP requests, filesystem behavior, source control, containers, or deployment flow.

## Videos

Videos in this section should provide clear conceptual explanations, visual walkthroughs, or conference talks that reinforce mental models used throughout the Blueprint.

## Blogs

Blogs in this section should be durable technical essays from credible practitioners, maintainers, engineering organizations, or educational publishers.

## Reference Material

Reference material in this section should include glossaries, protocol references, diagrams, cheat sheets, standards references, and compact explainers that learners can revisit while studying later chapters.
